## Continuous Integration

We use GitHub Actions for automated CI workflows.
The configuration file is stored in .github/workflows/ci.yml.
This setup helps us ensure that all commits are tested automatically when pushed.

## Branch Workflow

Each member works in a separate branch named after their StudentID-Name.

Branches are merged into main only through Pull Requests (PRs).

Any merge conflicts are resolved before changes are integrated into the main branch.
